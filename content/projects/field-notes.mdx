## Spatial Narrative

The project treats a building like a set of encounters: exterior approach, threshold, interior light, detail, landscape, and afterimage. Each section is built to hold one strong visual and one precise idea.

This is not a maximal portfolio template. It is a visual argument. The page gives the work enough order to be read, then gets quiet.

## Useful For

The system can support architecture studios, visualization artists, material researchers, interior concepts, and speculative spatial work where the story is larger than the render.
