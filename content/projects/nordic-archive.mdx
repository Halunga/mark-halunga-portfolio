## Brief

A portfolio has one job: make the work easy to understand quickly. This build is structured for a creative professional who needs selected projects, a writing index, an about page, and a direct contact path without burying visitors in decoration.

## Approach

The site uses Next.js, TypeScript, Tailwind, and MDX. Projects are stored as editable content files with metadata for cards, pages, SEO, and navigation. The layout stays strict: large type, direct hierarchy, fast pages, and mobile navigation that does not get in the way.

## Outcome

The result is a portfolio foundation that can grow without redesigning every page. New projects can be added by creating an MDX file, updating one import map, and replacing the image assets in `public/images`.
